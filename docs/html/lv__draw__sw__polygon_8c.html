Draw a polygon. Only convex polygons are supported

Parámetros
pointsan array of points
point_cntnumber of points
clip_areapolygon will be drawn only in this area
draw_dscpointer to an initialized lv_draw_rect_dsc_t variable

Check if the order of points is inverted or not. The normal case is when the left point is on y_min_i - 1 Explanation: if angle(p_left) < angle(p_right) -> inverted dy_left/dx_left < dy_right/dx_right dy_left * dx_right < dy_right * dx_left
