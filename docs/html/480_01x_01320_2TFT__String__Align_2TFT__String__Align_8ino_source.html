1/*
2Tests string alignment
3
4Normally strings are printed relative to the top left corner but this can be
5changed with the setTextDatum() function. The library has #defines for:
6
7TL_DATUM = 0 = Top left
8TC_DATUM = 1 = Top centre
9TR_DATUM = 2 = Top right
10ML_DATUM = 3 = Middle left
11MC_DATUM = 4 = Middle centre
12MR_DATUM = 5 = Middle right
13BL_DATUM = 6 = Bottom left
14BC_DATUM = 7 = Bottom centre
15BR_DATUM = 8 = Bottom right
16
17L_BASELINE = 9 = Left character baseline (Line the 'A' character would sit on)
18C_BASELINE = 10 = Centre character baseline
19R_BASELINE = 11 = Right character baseline
20
21So you can use lines to position text like:
22
23 tft.setTextDatum(BC_DATUM); // Set datum to bottom centre
24
25 Needs fonts 2, 4, 6, 7 and 8
26
27 #########################################################################
28 ###### DON'T FORGET TO UPDATE THE User_Setup.h FILE IN THE LIBRARY ######
29 ###### TO SELECT THE FONTS AND PINS YOU USE, SEE ABOVE ######
30 #########################################################################
31 */
