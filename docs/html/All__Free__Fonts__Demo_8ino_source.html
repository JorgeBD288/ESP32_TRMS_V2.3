1/*
2 Example for TFT_eSPI library
3
4 Created by Bodmer 31/12/16
5
6 This example draws all fonts (as used by the Adafruit_GFX library) onto the
7 screen. These fonts are called the GFX Free Fonts (GFXFF) in this library.
8
9 The fonts are referenced by a short name, see the Free_Fonts.h file
10 attached to this sketch.
11
12 Other True Type fonts could be converted using the utility within the
13 "fontconvert" folder inside the library. This converted has also been
14 copied from the Adafruit_GFX library.
15
16 Since these fonts are a recent addition Adafruit do not have a tutorial
17 available yet on how to use the fontconvert utility. Linux users will
18 no doubt figure it out! In the meantime there are 48 font files to use
19 in sizes from 9 point to 24 point, and in normal, bold, and italic or
20 oblique styles.
21
22 This example sketch uses both the print class and drawString() functions
23 to plot text to the screen.
24
25 Make sure LOAD_GFXFF is defined in the User_Setup.h file within the
26 library folder.
27
28 --------------------------- NOTE ----------------------------------------
29 The free font encoding format does not lend itself easily to plotting
30 the background without flicker. For values that changes on screen it is
31 better to use Fonts 1- 8 which are encoded specifically for rapid
32 drawing with background.
33 -------------------------------------------------------------------------
34
35 #########################################################################
36 ###### DON'T FORGET TO UPDATE THE User_Setup.h FILE IN THE LIBRARY ######
37 ###### TO SELECT YOUR DISPLAY TYPE AND ENABLE FONTS ######
38 #########################################################################
39*/
40
